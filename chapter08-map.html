8.1 Simple Hydrocarbons and Isomerism

Hydrocarbons ‚Ä¢ Naming ‚Ä¢ Isomers
üå±
Roots Core concepts & definitions
‚åÑ
  • Organic compounds contain carbon, usually with C‚ÄìH and/or C‚ÄìC bonding; may also include halogens, N, O, P, S.
  • Some carbon-containing compounds are not considered organic (e.g., carbonates, carbides, oxides of carbon).
  • Carbon has 4 valence electrons ‚Üí forms 4 covalent bonds, enabling chains, rings, and multiple bonds.

  • Structural isomers: same molecular formula, different connectivity ‚Üí different properties.
  • Geometric (cis‚Äìtrans) isomers: same connectivity, different spatial arrangement across C=C (restricted rotation).

General formulas (counting relationships; no units):

  • Alkane (acyclic, saturated): CnH2n+2
  • Cycloalkane (ring, saturated): CnH2n
  • Alkene (one C=C, unsaturated): CnH2n
  • Alkyne (one C‚â°C, unsaturated): CnH2n-2
  • Benzene: C6H6 (aromatic; delocalized electrons/resonance)

Naming branched alkanes (method):

  1. Find the longest continuous chain = parent.
  2. Number from the end closest to branches (lowest addresses).
  3. Name branches as alkyl (methyl, ethyl, ‚Ä¶); use di/tri/tetra for repeats; alphabetize branch names (ignore di/tri).
  4. Write: numbers + branch names + parent name; commas between numbers, hyphens between number-word.

Naming alkenes/alkynes: parent must contain the multiple bond; number from end closest to the multiple bond; suffix ‚Äú-ene‚Äù or ‚Äú-yne‚Äù; include position number.

Naming cycloalkanes: use ‚Äúcyclo-‚Äù; ring is parent if it has more carbons than any side chain; number ring to give lowest set of branch numbers; if ring is a branch, use ‚Äúcyclo‚Ä¶yl‚Äù.

Substituted benzene: mono: prefix + benzene. Poly: number ring for lowest combination; for two substituents: ortho(1,2), meta(1,3), para(1,4). Benzene branch = phenyl.

Leaves Examples, tips, pitfalls
‚åÑ
  • Drawing tip (skeleton formulas): every vertex/end is a carbon; add enough implied H so each carbon has 4 total bonds.
  • Isomer pitfall: ‚Äú3-methylbutane‚Äù is not a new molecule‚Äîrenumbering correctly gives ‚Äú2-methylbutane.‚Äù Always apply the lowest-number rule.

Combustion patterns:

  • Incomplete combustion: limited O‚ÇÇ ‚Üí CO and/or C(s) + H‚ÇÇO + energy (CO is toxic)
